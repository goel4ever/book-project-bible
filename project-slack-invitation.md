# Slack - Self Invite

##### Slack Channel to build this tool for

```
https://codeloom.slack.com
```

##### Features

```
Free to self invite
Abuse prevention implemented
Sign up only for restricted domains (https://codeloom.slack.com/admin/settings#signup_mode)
```
